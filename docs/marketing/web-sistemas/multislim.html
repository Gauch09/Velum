El sistema

Diseño y tecnología en línea

.MultiSlim combina instalación sencilla, bajo mantenimiento y eficiencia energética sobresaliente. Cada perfil define un paso (P) y un cuerpo (C) distintos: el ritmo de la fachada se elige como se elige una tipografía. Se monta sobre viga omega con parantes y costillas, o con fijación atornillada, suspendida, adhesiva o sobre sistema de placa de yeso.

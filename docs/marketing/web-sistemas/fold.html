El sistema

Una segunda piel de instalación simple

.Fold resuelve el revestimiento con tres componentes: muro o soporte, perfil omega y placa plegada. Sin subestructura pesada ni obra húmeda — ideal para combinar con construcción modular y en seco, proteger la mampostería y renovar frentes existentes con identidad propia.
